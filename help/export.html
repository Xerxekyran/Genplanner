Funktionalität :: Exportieren

Das Exportieren kann über den Menupunkt "File->Export selected solution" (im Deutschen: "Datei->Ausgewähltes Ergebnis exportieren") ausgeführt werden.


Datei-Menu um unter anderen einen Tourenplan zu exportieren

Es folgt ein Dialog zur Auswahl des Speicherorts. Exportiert wird eine XML-Datei, welche den aktuell ausgewählten Tourenplan repräsentiert. Die Auswahl der Tour geschieht über den Lösungs-Browser.
