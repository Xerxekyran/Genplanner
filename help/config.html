Konfigurationen :: config.xml

Die "config.xml" dient der Konfiguration des Prototyps. Das ändern kann mit jedem beliebigen Texteditor vorgenommen werden.

Konfigurationselement Mögliche Werte / Beschreibung
LogLevel Mögliche Werte: Debug, Info, Warn oder Error
Diese Einstellung beeinflusst, welche Nachrichten in die Logdatei geschrieben werden. Wird beispielsweise Info eingetragen, so werden keine Debug Meldungen mehr gespeichert.
LogFile Der Dateiname, in den die Logeinträge gespeichert werden sollen.
Language Mögliche Werte: en, de
Dieser Wert gibt an, welche Sprache standardmäßig beim Starten des Prototyps genutzt werden soll.
TravelCostWeight Zahlenmäßige Gewichtung der Fahrtkosten für die Bewertung. Höhere Werte bedeuten, dass der Algorithmus verstärkt versucht, die Gesamtstrecke der Touren zu minimieren.
VehicleLoadContraintWeight Zahlenmäßige Gewichtung der Ladevolumen-Einhaltung für die Bewertung. Höhere Werte bedeuten, dass der Algorithmus verstärkt versucht, die Ladevolumen der Fahrzeuge nicht zu überschreiten.
TimeWindowConstraintWeight Zahlenmäßige Gewichtung für das Einhalten von Kundenzeitfenstern bei der Bewertung. Höhere Werte bedeuten, dass der Algorithmus verstärkt versucht, die Kundenzeitfenster bei der Tourenplanung einzuhalten.
WorkTimeConstraintWeight Zahlenmäßige Gewichtung für das Einhalten von Arbeitszeiten der Mitarbeiter bei der Bewertung. Höhere Werte bedeuten, dass der Algorithmus verstärkt versucht, die Arbeitszeiten bei der Tourenplanung einzuhalten.
TournamentSelctionTournamentRounds Anzahl der "Runden" für das Turnierselektions-Verfahren. Standardmäßig wird nur eine Runde benutzt.
TournamentSelctionTournamentSize Anzahl von Lösungsvorschlägen für die Turnierselektion. Diese werden zufällig ausgewählt. Die beiden bestbewerteten von den Turnierteilnehmern werden für die Rekombination benutzt.
