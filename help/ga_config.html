Konfigurationen :: Genetischer Algorithmus

Der Genetische Algorithmus kann über den Dialog konfiguriert werden, welcher nach der Auswahl eines Testfalls erscheint:


Konfigurationsdialog für den Genetischen Algorithmus

Konfigurationselement Mögliche Werte / Beschreibung
Initialisation method Diese Einstellung entscheidet über das Initialisierungsverfahren.
Mögliche Werte:
  • Random: Eine zufällige Initialisierung der Touren. Die Kunden werden gleichmäßig auf die zur verfügung stehenden Fahrzeuge verteilt.
  • NearestFirst: Es wird zufällig ein Kunde ausgewählt und einem Fahrzeug zugeordnet. Anschließend werden geografisch in der nähe liegende Kunden der Tour hinzugefügt. Die Kunden werden gleichmäßig auf die zur verfügung stehenden Fahrzeuge verteilt.
Selection operator Diese Einstellung entscheidet über das Selektionsverfahren.
Mögliche Werte:
  • Random: Eine vollständig zufällige Auswahl von Lösungsvorschlägen.
  • Tournament: Es werden zufällig Lösungsvorschläge ausgesucht (in der Datei config.xml ist die Anzahl einstellbar). Die besten zwei Lösungsvorschläge werden genutzt.
Crossover operator Diese Einstellung entscheidet über das Rekombinationsverfahren.
Mögliche Werte:
  • BCRC: Best Cost Route Crossover. Zwei Routen werden zufällig ausgewählt und im jeweils anderen Elternteil gelöscht. Die fehlenden Kunden werden dann entsprechend der Bewertungsfunktion an der besten Stelle wieder eingefügt.
  • UOBX: Zwei Lösungen werden unter Verwendung einer zufälligen Schablone rekombiniert.
Natural selection method Diese Einstellung entscheidet über das Ersetzungsschema.
Mögliche Werte:
  • BestChromeSelector: Elitismus. Die besten Lösungen werden in die nächste Generation übernommen.
  • WeightedRoulette: Jede Lösung erhält eine seiner Bewertung proportionale Chance, in die nächste Generation übernommen zu werden.
Population amount Anzahl der gleichzeitig existierenden Lösungsvorschläge. Diese bilden die Population und somit die genetische Vielfalt um neue Lösungen zu erzeugen.
Maximum of generations Dies ist das Abbruchkriterium. Nach Erreichen der hier eingestellten Generation, wird der Algorithmus automatisch auf pausiert gestellt.
