Funktionalität :: Starten / pausieren der Logistikplanung

Ist bereits ein Testfall eingeladen, kann der Optimierungsvorgang pausiert und wieder fortgesetzt werden. Das Starten kann mit dem Menupunkt "File->Start calculation" (im Deutschen: "Datei->Berechnung starten") ausgeführt werden. Wurde die Berechnung aufgrund des Abbruchkriteriums beendet, so kann diese ebenfalls wieder gestartet werden. Die maximale Anzahl von Generationen wird dabei verdoppelt und die Berechnung geht von der aktuellen Generation aus weiter.

Das Pausieren erfolgt durch die Auswahl des Menupunktes "File->Pause calculation" (im Deutschen: "Datei->Berechnung pausieren"). Der Algorithmus wird dann, nachdem die Berechnungen für die aktuellen Generation ausgeführt wurden, pausieren.
